

# FINAL TARGET

The site should feel like:

* Awwwards portfolio
* Creative AI studio
* Premium startup landing page
* Interactive cinematic experience

Not:

* student portfolio
* template website
* basic React page

---

# MOST IMPORTANT RULE

Every section should have:

1. A focal point
2. Motion
3. Depth
4. Visual hierarchy
5. One memorable element

If a section lacks these, it will feel flat.
